## Workflow

### Step 1 — Intake questions

Ask all of the following in a single numbered list. Wait for the user to answer before proceeding.

1. What is the article topic or working title?
2. What is the primary keyword to target?
3. Where should the TrueProfit link go? (Homepage / Shopify App listing / specific blog URL — provide the URL if other)
4. What anchor text should be used for the TrueProfit link(s)? (If not provided, determine appropriate anchor text based on topic and context)
5. Which format is this guest post?
   - **Option A: Normal SEO post** — editorial blog style, practitioner voice, optimized for search
   - **Option B: Press release** — formal, brand-narrated, structured for PR distribution
6. Target word count? (Default: 1,500–2,500 words)
7. Do you have a reference article from the publisher to match their style? (Paste a URL or sample text — required for Option A, optional for Option B)

Once the user answers, move to Step 2.

---

### Step 2 — Analyze reference material (if provided)

If the user provides a reference article or URL, extract and match:

- **Tone**: casual/conversational, neutral/editorial, formal/corporate
- **Opening structure**: hook, stat-led, question, problem statement, or direct assertion
- **Heading format**: H2 only, H2+H3, numbered, plain
- **Body structure**: numbered list of ideas, flowing sections, step-by-step, comparison tables
- **Internal link style**: always use inline hyperlinks on descriptive text — never "Further Reading" callouts, footnotes, or bare URLs
- **List style**: bullet with bold label + colon, plain bullets, numbered, or no lists
- **Paragraph length**: short (1–3 sentences), medium (3–5 sentences), or long blocks
- **CTA/pitch placement**: end of article, mid-article, or dedicated section
- **Closing**: summary + CTA, open-ended, or media contact block (press release)
- **Length**: approximate word count of reference

If no reference is provided for Option A, default to TrueProfit native blog style (see below).

---

### Step 3 — Write the article

#### Option A: Normal SEO Post

Follow **TrueProfit native blog style** unless the reference article overrides specific elements.

**TrueProfit native blog style rules:**
- Voice: first/second person, practitioner, peer-to-peer — write like someone who has done this, not a brand talking at you
- Tone: direct, opinionated, no hedging, no corporate filler
- Opening: 2–3 short paragraphs that immediately establish what the article covers and who it is for. No "In today's fast-paced world" openings.
- Quick Recap box: place after the intro — a 3–5 bullet summary of what the reader will learn or the key takeaways. Label it **Quick Recap:**
- Headings: H2 for main sections, H3 for subsections. Numbered headings for list-style articles (e.g. "1. Dropshipping", "2. Print-on-demand").
- Body: short paragraphs (2–4 sentences max). No walls of text.
- Lists: use bullet points only when listing 3+ parallel items. Each bullet at least one full sentence.
- Tables: use for comparisons (pricing tiers, plan features, cost breakdowns).
- Links: always use inline hyperlinks on descriptive text. Never use "Further Reading" callouts or bare URLs.
- No em dashes. Use commas, colons, or semicolons instead.
- No AI-sounding filler: no "delve into", "it's worth noting", "in conclusion", "in summary", "it is important to", "game-changer", "revolutionize".
- Bold only for genuinely important terms or labels, not for decoration.
- Benchmarks and data: if the article includes statistics or benchmarks about Shopify, dropshipping income, profit margins, or ecommerce performance, always use TrueProfit's own numbers from the Brand Reference and published TrueProfit data. Never fabricate stats or pull generic figures from unattributed sources.
- TrueProfit pitch: place as a dedicated section near the end. Use a natural transition from the article's topic to why profit tracking matters. Feature list uses plain bullets. Include a dashboard image placeholder labeled: *[TrueProfit Dashboard]*
- CTA: short, direct call-to-action box or paragraph after the pitch section.
- Meta description: place at the very top of the document, labeled **Meta description:** (~155 characters, includes primary keyword).

**SEO structure template (adapt as needed):**
```
Meta description: [~155 chars]

# [H1 Title with primary keyword]

[Intro: 2–3 short paragraphs]

Quick Recap:
- [bullet]
- [bullet]
- [bullet]

## [H2 Section 1]
[body]

## [H2 Section 2]
[body]

... (main body sections)

## [TrueProfit pitch section]
[2–3 para intro + feature list + dashboard image placeholder + CTA]
```

#### Option B: Press Release

Follow formal PR/press release conventions.

**Press release style rules:**
- Voice: third-person brand narration throughout. Refer to TrueProfit as "TrueProfit" or "the company", not "we" or "you".
- Tone: professional, authoritative, neutral. No casual language.
- Opening: strong headline + subheadline. First paragraph answers who, what, when, where, why in 2–3 sentences.
- Structure: labeled H2 sections (Introduction, [Topic Section], Key Features / Key Metrics, [Optional: Merchant Success Stories or Expert Commentary], Looking Ahead, Conclusion)
- No Quick Recap box, no FAQ section, no numbered product lists.
- Feature lists use plain bullets with **Bold label:** description format.
- Paragraphs: medium length (3–5 sentences). More formal than blog style.
- Links: hyperlink TrueProfit name or specific feature terms inline.
- Closing: Conclusion paragraph + Media Contact block:
  ```
  Media Contact
  Company Name: TrueProfit
  Contact Person: TrueProfit Team
  Email: partners@trueprofit.io
  City: Ho Chi Minh City
  Country: Vietnam
  Website: https://trueprofit.io
  ```
- No meta description at the top.
- No FAQ at the end.
- Image placeholders labeled: *[Thumbnail link]* and *[Image link]* where relevant.

## Link and anchor text rules

- Always hyperlink TrueProfit on first mention using the URL provided by the user (default: https://trueprofit.io)
- If the user did not specify anchor text, determine it contextually:
  - Homepage link: use "TrueProfit" or a descriptive phrase like "net profit analytics for Shopify"

- Never use generic anchors like "click here", "this tool", or "learn more" as the only anchor
